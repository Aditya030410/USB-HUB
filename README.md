# USB HUB
- The Files uploaded are for my custom USB HUB which can be reviewed on any PCB AND SCHEMATIC making platform. It has an Upstream port of C-Type and 4 downstream ports, 2 are A-type and 2 are C-type. It can be used for a laptop to connect more USB devices allowing u to connect upto 4 more devices !!
